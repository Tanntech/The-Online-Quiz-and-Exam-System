1. What will be the output of the following code?
class Test {
    static {
        System.out.print("A ");
    }
    {
        System.out.print("B ");
    }
    Test() {
        System.out.print("C ");
    }
    public static void main(String[] args) {
        new Test();
    }
}


a) A B C
b) B C
c) A C
d) Compilation error

Correct Answer: a

Explanation:
Static block executes first, then instance block, then constructor.

3. What will be the output?

    void show() {
        System.out.print("Parent ");
    }
}

    void show() {
        System.out.print("Child ");
    }
}
public class Demo {
    public static void main(String[] args) {
        Parent p = new Child();
        p.show();
    }
}


a) Parent
b) Child
c) Compilation error
d) Runtime error

Correct Answer: b

Explanation:
Runtime polymorphism: method call depends on object type, not reference.
