1. Which part of an assembly uniquely identifies its version and culture?

A. IL code
B. Metadata tables
C. Assembly manifest
D. Resources

Correct Answer: C
Explanation: The assembly manifest stores version, culture, and strong name information.

2. Which tool is used to inspect IL code of a .NET assembly?

A. CLR Profiler
B. ILDASM
C. NuGet
D. MSBuild

Correct Answer: B
Explanation: ILDASM (Intermediate Language Disassembler) displays IL and metadata.

3. Which type of code runs under CLR control?

A. Native code
B. Unmanaged code
C. Managed code
D. Unsafe code

Correct Answer: C
Explanation: Managed code executes under CLR with services like GC and security.

4. Which constructor is executed first in an inheritance chain?

A. Derived class constructor
B. Base class constructor
C. Static constructor
D. Object initializer

Correct Answer: B
Explanation: Base class constructors execute before derived class constructors.

5. What is the purpose of a static constructor?

A. Initialize instance members
B. Release resources
C. Initialize static members
D. Overload methods

Correct Answer: C
Explanation: Static constructors initialize static data and run once per type.

6. Which access modifier allows access only within the same assembly?

A. public
B. private
C. protected
D. internal

Correct Answer: D
Explanation: internal limits access to the same assembly.

8. Which keyword ensures a struct cannot be inherited?
