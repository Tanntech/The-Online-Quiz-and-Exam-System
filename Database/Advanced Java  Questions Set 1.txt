1. What is the main advantage of JSP over Servlets?

A. JSP runs faster than Servlets
B. JSP separates presentation from business logic
C. JSP replaces the need for Java
D. JSP cannot use Java code

Correct Answer: B
Explanation: JSP focuses on presentation, while Servlets handle business logic, improving maintainability.

2. Which method handles POST requests in HttpServlet?

A. doGet()
B. service()
C. doPost()
D. init()

Correct Answer: C
Explanation: doPost() processes HTTP POST requests.

3. What does @Autowired do in Spring?

Correct Answer: B
Explanation: @Autowired performs dependency injection.

4. Which annotation defines a REST controller?

A. @Controller
