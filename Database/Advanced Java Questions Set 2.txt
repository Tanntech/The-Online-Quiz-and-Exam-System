1. Which of the following is not true for servlet?

A. It is persistent
B. Platform independent
C. It is multi threaded
D. It is single threaded

Correct Answer: D
Explanation: Servlets are multi-threaded by default, not single-threaded.

2. Which method of the servlet is called several times in its life?

A. init()
B. doPost()
C. destroy()
D. None of above

Correct Answer: B
Explanation: doPost() is invoked every time a POST request is received.
