1. What will be the output?
int x = 10;
int *p = &x;
int **q = &p;
cout << **q;

A) Address of x
B) Address of p
C) 10
D) Garbage

Correct Answer: C) 10
Explanation: q points to p, *q gives p, and **q gives value of x.

2. What is the output?
static int count = 0;
void fun() {
    count++;
}
int main() {
    fun(); fun();
    cout << count;
}


A) 0
B) 1
C) 2
D) Error

Correct Answer: C) 2
Explanation: Static variable retains its value between function calls.

3. Which of the following is true about inline functions?

A) Always replaced by macro
B) Function call overhead is removed
C) Cannot access static variables
D) Cannot be recursive

Correct Answer: B) Function call overhead is removed
Explanation: Inline functions reduce function call overhead by replacing the call with function body (compiler decision).

5. What is the output?
