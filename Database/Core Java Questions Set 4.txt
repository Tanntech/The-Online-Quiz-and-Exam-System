1. What will be the output of the following code?
class Test {
    static int x;
    public static void main(String[] args) {
        System.out.println(x);
    }
}


a) Garbage value
b) Compilation error
c) 0
d) Runtime error

Correct Answer: c

Explanation:
Static variables get default values. For int, the default value is 0.

3. What is the output of the following program?
class Demo {
    Demo() {
        System.out.print("A ");
    }
    Demo(int x) {
        this();
        System.out.print("B ");
    }
    public static void main(String[] args) {
        new Demo(5);
    }
}


a) A
b) B
c) A B
d) Compilation error

Correct Answer: c

Explanation:
this() calls the default constructor first, then prints B.
