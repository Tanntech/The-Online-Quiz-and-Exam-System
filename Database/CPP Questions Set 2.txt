1. Which keyword prevents a class from being inherited?
A) final
B) static
C) const
D) sealed
Correct Answer: A
Explanation: final prevents further inheritance.

2. What is the size of an empty class in C++?
A) 0
B) 1
C) Depends on compiler
D) 4
Correct Answer: B
Explanation: Minimum size is 1 byte to ensure unique addresses.

3. Which cast is safest in C++?
A) C-style cast
B) reinterpret_cast
C) static_cast
D) dynamic_cast
Correct Answer: D
Explanation: dynamic_cast performs runtime type checking.

4. What does mutable allow?
A) Modify const object
B) Modify static data
C) Modify constant variable
D) Modify enum
Correct Answer: A
Explanation: Mutable members can change in const objects.

5. Which container does NOT allow duplicate values?
A) vector
B) list
C) set
D) deque
Correct Answer: C
Explanation: set stores unique elements.

6. Which operator cannot be overloaded?
A) +
B) ::
C) []
D) <<
Correct Answer: B
Explanation: Scope resolution operator cannot be overloaded.

7. What happens if delete is used instead of delete[]?

B) Runtime error

Explanation: Must match allocation type.

8. Which keyword is used to handle exceptions?

9. Which STL container provides fastest search?

10. Which header is required for std::sort()?
