1. Which JDBC interface is used to execute SQL queries that return multiple results?

A. Connection
B. Statement
C. PreparedStatement
D. ResultSet

Correct Answer: B
Explanation: Statement executes SQL queries; ResultSet holds the results.

2. What is the default method in a servlet to handle HTTP POST requests?

A. service()
B. doPost()
C. doGet()
D. init()

Correct Answer: B
Explanation: doPost() is called for HTTP POST requests.

3. Which JSP implicit object is used to get initialization parameters?

A. application
B. config
C. session
D. request

Correct Answer: B
Explanation: config represents ServletConfig and allows access to init parameters.

4. Identify the Hibernate annotation for mapping a class to a database table:
