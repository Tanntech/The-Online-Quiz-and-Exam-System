1. What is the output of the following code?
int x = 5;
int &y = x;
y++;
cout << x;


A) 4
B) 5
C) 6
D) 0

Correct Answer: C) 6
Explanation: y is a reference to x. Incrementing y also increments x.

2. What is the size of an empty class in C++?

A) 0
B) 1
C) Depends on compiler
D) Cannot be determined

Correct Answer: B) 1
Explanation: Empty classes have size 1 byte to ensure each object has a unique address.

3. Which of the following is true about copy constructors?

A) Automatically called during object assignment
B) Cannot be overloaded
C) Initializes object with another object
D) Always private

Correct Answer: C) Initializes object with another object
Explanation: Copy constructor creates a new object as a copy of an existing object.

4. What will the following code print?
class A {
public:
    A() { cout << "A"; }
    ~A() { cout << "~A"; }
};
int main() {
    A a1, a2;
}


A) A~A
B) AA~~
C) AA~A~
D) A

Correct Answer: C) AA~A~
Explanation: Two objects constructed (AA), destructors called in reverse order (~A~).

5. Which of the following statements about virtual functions is true?

A) Can be static
B) Enables compile-time polymorphism
C) Enables runtime polymorphism
D) Cannot be inherited

Correct Answer: C) Enables runtime polymorphism
Explanation: Virtual functions allow derived class methods to be called through base class pointer at runtime.
