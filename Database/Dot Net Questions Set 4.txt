1. Which component of .NET converts IL code into native machine code at runtime?

A. CLR
B. JIT Compiler
C. CTS
D. CLS

Correct Answer: B
Explanation: The Just-In-Time (JIT) compiler converts Intermediate Language (IL) into native machine code during execution.

2. What is the main purpose of the Common Type System (CTS)?

Explanation: CTS defines how data types are declared, used, and managed to ensure type safety across .NET languages.
