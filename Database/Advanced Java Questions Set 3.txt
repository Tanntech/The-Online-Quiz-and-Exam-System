1. Which annotation is used to define a Spring service layer component?

A. @Component
B. @Repository
C. @Service
D. @Controller

Correct Answer: C
Explanation: @Service is used to define business logic components.

2. What does @Repository provide in Spring?

A. Security handling
B. Transaction management and exception translation
C. UI rendering
D. REST mapping

Correct Answer: B
Explanation: @Repository translates persistence exceptions into Spring exceptions.

3. Which annotation maps HTTP GET requests?

C. @GetMapping
D. @FetchMapping

Correct Answer: C
Explanation: @GetMapping is a shortcut for GET requests.
